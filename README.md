# 🏥 Healthcare Analytics Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze healthcare data across multiple dimensions, including patients, doctors, hospital operations, and financial performance. It enables healthcare organizations to monitor key metrics and improve decision-making through data-driven insights.

---

## 🎯 Objectives

* Track patient admissions and treatment trends
* Analyze doctor performance and workload
* Monitor hospital resource utilization
* Evaluate financial performance and revenue streams

---

## 📊 Dashboard Modules

### 📍 Overview

* Summary of key KPIs
* Total patients, doctors, revenue, and occupancy
* High-level performance insights

---

### 👥 Patients

* Patient demographics (age, gender)
* Admission and discharge trends
* Disease/diagnosis analysis
* Patient distribution over time

---

### 🩺 Doctors

* Doctor-wise patient handling
* Performance metrics
* Department-wise analysis
* Workload distribution

---

### 🏨 Hospital

* Bed/room availability and utilization
* Department performance
* Admission rates
* Resource management insights

---

### 💰 Finance

* Total revenue analysis
* Treatment costs
* Billing trends
* Profitability insights

---

## 🛠 Tools & Technologies

* Power BI
* SQL (Data Analysis)
* Excel (Data Cleaning)

---

## 💡 Key Insights

* Peak patient admissions during specific periods
* Uneven workload distribution among doctors
* Certain departments generate higher revenue
* Resource utilization can be optimized

---

## 🚀 Business Impact

* Improves hospital resource planning
* Enhances patient management
* Supports financial decision-making
* Optimizes doctor workload distribution
